# Docker 란?

> 컨테이너 기반의 오픈소스 가상화 플랫폼

#### 도커 컨테이너
컨테이너 : 무역에서 나온 단어로, 배에 실을 화물의 운송 단위를 규격화 한것이다.
         포워더(브로커)들이 다양한 소스의 화물들을 한 컨테이너에 모아서 보내기도 한다
도커 컨테이너 : 다양한 프로그램, 실행환경을 컨테이너로 추상화하고 동일한 인터페이스를 제공해준다.
